標題和標籤

#的數量代表標題的級別，共6級。也可以在文本下一行加入任意數量的 ==（一級標題） 或 -- （二級標題）。 標題在#後加上空格 ，否則會變成標籤。 標籤內使用斜槓 / 可實現標籤嵌套。 #大標籤/二標籤 #大標籤/二標籤2

強調

Bold: 前後加兩個星號或下劃線。 Italic: 前後加一個星號或下劃線。 Bold & Italic: 前後加三個星號或下劃線。 ==Highlight==: 前後加兩個等號。

引用

段落前添加 > 符號，多個段落引用時於段落間空白也添加 > 。

塊引用可嵌套，添加 » ，也可以部分地支持 Markdown。

列表

  1. 有序列表用數字 1+英文句點起始，但之後可以不按數學順序排列。
  2. Shift + Enter 可以， 另起一行但不進入下個列表。
  • 無序列表可以前置 - , * , + 表示。
    1. 縮進一個或多個列表可創建嵌套列表。
    2. 在保留列表連續性的情況下插入另一種元素，將其縮進四個空格或一個製表符。 ^56c0d0
    • 代碼塊通常縮進 4 空格或 1 製表符，放入列表時，請縮進兩倍。
  • 有時 Shift + Tab 無法解決問題，可以用 Ctrl + 「[」 或 「]」 。

代碼

包裹在反引號「`」中。若其中已包含反引號，則代碼包裹在雙反引號中轉義。使用三個反引號（或縮進，但好像不行）來表示代碼塊，還可以在其後添加語言標識符（加在首個三重引號之後）。

def hello()
print('hello world')

超鏈接

[超鏈接顯示名] + （超鏈接地址“超鏈接 title”），title 指光標懸停時出現的文字。 尖括號能使得 URL 和 Email 地址可點擊。 https://markdown.com.cn 引用類型鏈接分兩部分：

  1. 第一部分爲應顯示爲鏈接的文本，格式爲[文本][標籤]，標籤包含 case-insensitive 的字母、數字、空格或標點符號。如： 嗶哩嗶哩 此處標籤可重複使用，進而指向同一鏈接。
  2. 第二部分，[標籤]:<url>'可選標題’ ，其中 url 也可以不用尖括號，標題可以置入括號、單引號或雙引號。兩部分似乎不能並列。第二部分可放在文檔中任意位置，但要自成段落（用空行隔開，不可接在正文後面）。 網頁鏈接變量 的格式：
  3. 首先輸入
    • [顯示文本內容] + [變量名]
      • 變量名可以自己取，沒啥限制，任意字符都可以
  4. 在文檔任意一個區域，輸入：
    • [變量名] + : + 空格 + 鏈接地址 （這個空格 不打也沒事)

圖片

![替代文本](圖片鏈接 ‘可選圖片 title’)。如給其增加鏈接，則將上述內容置入方括號，並跟隨一個置入圓括號的鏈接。 除了手寫 Markdown，插入圖片方法還有：

  1. 拖放圖片。
  2. 使用快捷鍵：在編輯模式下按下 Ctrl + Shift + I 打開文件選擇對話框。
  3. 使用剪貼板。 Obsidian 的圖片設置大小是用 | 分隔，後面寫寬度數值，單位默認是 px
    設定好寬度，高度會自動等比例調整

改文字顏色就很麻煩，需要用到 HTML 。 或這樣 可以使用插件「Editing Toolbar」解決，但真的不方便。富文本不是 Markdown 的強項。

內鏈跳轉

輸入 [[ 之後就知道了。比如 [[#^4b30cf|點我跳轉伊織萌]] 此外：

### 標題但是重名 blabla 
### 標題但是重名 ^title2 

blabla2 --- [跳轉測試](#標題但是重名)

Markdown 擴展語法

表格

使用 pipe (’ | ‘) 創建表格，使用三個或更多連字符（’ _’）創建 title 。 通過在標題行的連字符左側、右側或兩端添加冒號，可以將該列文本對齊到左右側或居中。 部分格式可以應用於表格中。表格同樣需要獨立的一段。 表格中的換行可以使用 HTML 標籤 <br>。

標題一 標題二
一 二

腳註

使用 [^標識符] 這樣的格式創造腳註參考，並在其他地方使用 [^標識符]: 文本 來添加腳註。 標識符無論如何編寫，腳註在輸出中都按順序編號；腳註無論位於何處，輸出中都會位於文檔末尾。1。 縮進可以把不同段落添加進同一條腳註。

標題編號

使用 Markdown 的 HTML 注釋語法 <!-- -->，在標題前添加一個自定義 ID。例如：<!-- custom-id -->。 你可以使用比如雙重方括號鏈接語法來引用具有自定義 ID 的標題：

刪除線和下劃線

刪除線使用波浪號，下劃線使用 html 的<u></u>

任務列表

- + Space + [] + 列表內容 ，方括號裏有空格，在方括號內輸入任意字符會打勾，如果輸入 x 會額外添加刪除線，但是會把方括號和內容之間的空格也打上刪除線，需要在末尾補個空格。
